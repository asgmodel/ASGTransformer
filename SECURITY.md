# Security Policy

## Intended scope

ASGTransformer is intended for authorized defensive cybersecurity training, awareness, tabletop exercises, incident-response preparation, and control validation.

## Prohibited use

Do not use this project to facilitate unauthorized access, malware deployment, credential theft, evasion, destructive activity, or operational exploitation.

## Reporting a vulnerability

Report security issues privately to the project maintainers. Include the affected version, reproduction conditions, impact, and a minimal non-destructive proof of concept. Do not disclose unresolved vulnerabilities publicly.

## Model-output handling

Treat generated scenarios as untrusted draft content. Apply human review, authorization controls, logging, and environment isolation before operational use.
